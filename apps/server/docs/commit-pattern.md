# Commit Pattern

## 📌 Conventional Commits

Formato:

tipo: descrição

## Tipos

- feat → nova funcionalidade
- fix → correção
- docs → documentação
- chore → config
- refactor → melhoria de código

## Exemplos

feat: add login route
fix: correct auth middleware
docs: update backend guide

## 🧠 Boas práticas

- mensagens curtas
- em inglês (opcional)
- foco no que mudou
